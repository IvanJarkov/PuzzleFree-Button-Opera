# PuzzleFree Button (Opera)

A lightweight browser extension that adds a toolbar button for quick access to PuzzleFree.Game.

## What it does
- Adds a single toolbar button
- Opens https://puzzlefree.game in a new tab
- No tracking, no data collection, no background activity

## Build instructions
This extension does not require a build process.

### Environment
- OS: Any
- No build tools required (no Node.js, npm, or bundlers)

### Steps
1. Clone or download this repository
2. The extension package is created by directly zipping the source files:
   - manifest.json
   - background/service worker script
   - icon files
3. Upload the ZIP archive to the Opera Add-ons dashboard
